🍴 Forkify

Forkify is a recipe search application that lets you discover, view, and bookmark recipes from a database of over 1,000,000 recipes.
Search for any dish
adjust serving sizes on the fly, and save your favorites for later.

✨ Features
-📖 View recipe details — ingredients, cooking time, servings, and preparation info
-📱 Responsive design — works across desktop, tablet, and mobile

🛠️ Built With
-HTML5
-Tailwind CSS
-JavaScript (ES6+ modules)
-Font Awesome for icons
-Forkify API for recipe data
